# 高校社团学生会管理系统 python1203

## 项目概述

本项目是基于 Python 开发的高校社团学生会管理系统，旨在提供一套完善、高效的社团活动和信息管理解决方案。

## 技术栈

### 后端

- 开发语言：Python
- 框架：Django
- Python 版本：3.7.7
- 数据库：MySQL 5.7+
- 数据库工具：Navicat 11+
- 开发软件：PyCharm

### 前端

- NodeJS
- Vue.js
- HTML

## 功能模块

### 系统角色

- 管理员
- 用户

### 管理员功能模块

管理员可执行以下操作：

1. 会员管理
2. 留言板管理
3. 活动录入
4. 社团报名管理
5. 社团活动添加
6. 社团信息管理

### 用户功能模块

用户可执行以下操作：

1. 社团活动添加
2. 社团信息录入
3. 活动信息管理
4. 社团报名管理
5. 留言板管理
6. 社团信息管理

## 运行环境

- 后端：Python 3.7.7、Django、MySQL 5.7+
- 前端：NodeJS、Vue.js、HTML
